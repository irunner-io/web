Docs/Reference/API endpoints

API endpoints

HTTP endpoints exposed by the iRunner control plane. Internal endpoints are only accessible within the VPC via the internal ALB. The public ALB only forwards the webhook path; all other paths return 403.

Internal endpoints

These endpoints are accessible from within the VPC at http://control-plane.<prefix>.internal:8080.

GET /health

Returns HTTP 200 if the control plane is healthy. Used by ALB target group health checks.

Terminal
curl -s http://control-plane.ir-west.internal:8080/health
Response (200 OK)
OK

GET /internal/status

Returns JSON with license information, pool status summary, and control plane version.

Terminal
curl -s http://control-plane.ir-west.internal:8080/internal/status | jq .
Response (200 OK)
{
  "version": "0.5.4",
  "license": {
    "customer": "acme-corp",
    "plan": "enterprise",
    "expires_at": "2027-01-15T00:00:00Z",
    "max_concurrency": 500,
    "current_concurrency": 12
  },
  "pools": {
    "total": 4,
    "healthy": 4
  },
  "uptime_seconds": 86412
}

GET /metrics

Prometheus-compatible metrics endpoint. Exposes pool utilization, job latency histograms, and queue depth gauges.

Terminal
curl -s http://control-plane.ir-west.internal:8080/metrics
Response (200 OK, text/plain)
# HELP ir_pool_active_instances Current active instances per pool
# TYPE ir_pool_active_instances gauge
ir_pool_active_instances{pool="large-ec2"} 5
ir_pool_active_instances{pool="ecs-default"} 3

# HELP ir_job_duration_seconds Job execution duration
# TYPE ir_job_duration_seconds histogram
ir_job_duration_seconds_bucket{pool="large-ec2",le="60"} 12
ir_job_duration_seconds_bucket{pool="large-ec2",le="120"} 45
ir_job_duration_seconds_bucket{pool="large-ec2",le="300"} 89

# HELP ir_queue_depth Current SQS queue depth
# TYPE ir_queue_depth gauge
ir_queue_depth{queue="jobs"} 2
ir_queue_depth{queue="events"} 0

Public endpoint

POST /github/webhook

GitHub webhook receiver. Validates the HMAC-SHA256 signature using the configured webhook secret. Only processes workflow_job events.

Terminal
# GitHub sends this automatically; manual testing example:
curl -X POST https://webhook.ir-west.example.com/github/webhook \
  -H "Content-Type: application/json" \
  -H "X-GitHub-Event: workflow_job" \
  -H "X-Hub-Signature-256: sha256=abc123..." \
  -d @payload.json
Response (200 OK)
{"accepted": true}

Default deny

All paths not listed above return HTTP 403 via the ALB default action. This applies to both the public and internal ALB listeners for unrecognized routes.

Terminal
curl -s https://webhook.ir-west.example.com/anything-else
Response (403 Forbidden)
{"error": "forbidden"}
