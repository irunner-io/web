CLI reference

Command-line tools shipped with iRunner: the control plane binary and the license generation utility.

ir

The iRunner control plane server. Runs as an ECS service (or standalone binary) and manages pool orchestration, webhook processing, and runner lifecycle.

Flags

FlagDefaultDescription
--config—Path to config.yaml. Accepts a local file path or an S3 URI (e.g. s3://bucket/config/config.yaml).
--port8080Port the HTTP server listens on.
--log-levelinfoLog verbosity: info, debug, warn, error.

Environment variables

VariableDescription
IR_CONFIG_PATHEquivalent to --config. Flag takes precedence if both are set.
IR_LOG_LEVELEquivalent to --log-level. Flag takes precedence.
IR_LICENSE_KEYSigned license JWT. Required for production deployments.
IR_CACHE_URLURL of the cache service endpoint (e.g. http://cache.ir-west.internal:9090). Injected into runner environment.
AWS_REGIONAWS region override. Defaults to instance metadata or ECS task region.

Usage

Terminal
# Start with S3 config
ir --config s3://ir-west-releases-111111111111/config/config.yaml

# Start with local config and debug logging
ir --config /etc/ir/config.yaml --log-level debug

# Start using environment variables
export IR_CONFIG_PATH=s3://ir-west-releases-111111111111/config/config.yaml
export IR_LICENSE_KEY=eyJhbGciOiJFZERTQSIs...
ir

ir-license-gen

License key generation tool. Used to create signed license JWTs for customer deployments. Typically run by the iRunner team, not end users.

Flags

FlagDescription
--gen-keysGenerate a new Ed25519 keypair and write to private.pem / public.pem.
--private-keyPath to the Ed25519 private key for signing.
--customerCustomer identifier embedded in the license.
--planLicense plan: trial, team, or enterprise.
--termLicense validity period: 14d, 1m, 3m, 6m, 12m.
--max-concurrencyMaximum concurrent jobs allowed by this license.

Usage

Terminal
# Generate a new keypair
ir-license-gen --gen-keys

# Generate a trial license (14 days, 10 concurrent jobs)
ir-license-gen \
  --private-key private.pem \
  --customer acme-corp \
  --plan trial \
  --term 14d \
  --max-concurrency 10

# Generate an enterprise license (12 months, 500 concurrent jobs)
ir-license-gen \
  --private-key private.pem \
  --customer bigcorp \
  --plan enterprise \
  --term 12m \
  --max-concurrency 500
Tip

The generated license JWT is printed to stdout. Set it as the IR_LICENSE_KEY environment variable in your ECS task definition or pass it via Secrets Manager.
